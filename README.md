# Real Estate Market Analysis in Maricopa County, Arizona
Final Capstone Project for Data Analytics Course

Overview
The objective of this project is to analyze housing market conditions in Maricopa County, with a particular emphasis on affordability and neighborhood level disparities. Maricopa County offers neighborhoods across the spectrum in a relatively small area so will offer some insights not able to be gained in other areas. 

Technologies used
Power BI
jupyter notebook
microsoft word
home computer

How to reproduce the analysis
retrieving the current data from PolicyMap and runing through the steps in the included notebooks will clean the data
model it for analysis and run it through a linear regression model

Data used is in the datasets directory in .csv files with the final report in the documentation directory as a pdf
the jupyter notebooks used are in the scripts directory, one has visualizations. The power BI file in in the visualizations directory.
